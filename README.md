# Кейс-задача №3  
## Класс WORKER (на базе Университета «Синергия»)

## Описание проекта

Проект представляет собой реализацию класса `WORKER` на языке Python в рамках учебного задания по объектно-ориентированному программированию.

Класс предназначен для хранения и обработки данных о работниках организации (на примере Университета «Синергия»).

Программа выполняет:
- ввод данных работников с клавиатуры;
- вычисление стажа работы;
- вывод работников, чей стаж превышает заданное значение;
- вывод сообщения при отсутствии подходящих работников.

## Описание класса WORKER

### Атрибуты

- `surname` — фамилия работника  
- `name_initial` — инициал имени  
- `patronymic_initial` — инициал отчества  
- `position` — занимаемая должность  
- `salary` — заработная плата  
- `year_hired` — год поступления на работу  

### Методы

- `from_input()` — создание объекта на основе пользовательского ввода  
- `experience_years()` — вычисление стажа работы  
- `is_experience_gt()` — проверка превышения заданного стажа  
- `get_short_name()` — формирование ФИО в формате «Фамилия И.О.»  
- `display()` — вывод полной информации о работнике
